# attendanceScript   
自动统计考勤python脚本  
创建背景：网课时期，对于腾讯课堂导出详细考勤记录表需要做一个统计，这种机械化操作当然就需要程序解决√

* today()：考勤表
* peoples()：成员表
* judge()：主要函数，匹配考勤表内成员与成员表成员，进行一系列操作
